About Code Fiesta

Code Fiesta is a one week hackathon that targets tech enthusiasts

Our focus is on improving learning, innovation and building systems using latest technologies and resources

There are basically three(3) cycles of the Code Fiesta and this holds quarterly on an affixed date

The interested applicants of this Fiesta will register before hand and take some short test after which selection and invitation is sent out to those successful. After the selection some guidelines on how the program for that cycle is going to go will be communicated to the participants. These guidelines include likely projects to be embarked on and technologies to be used, programming language involved, and level of technical know how.

Why join Code Fiesta?

Learn
- You learn more about recent technologies and connect with other developers
Innovate
- You can share ideas on projects to build to aid the community
Educate
- Education is the key thing. it equips you greater task ahead

Our Goals

  • To build a tech community starting from the East
  • To connect developers from all works of life
  • To enlighten the surrounding community on the latest technology
